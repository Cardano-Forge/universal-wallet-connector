## Contributing

1. Fork the project
2. Create a Feature Branch
3. Commit your changes
4. Push your changes
5. Create a PR

### Local Development

*Install and start the development server*

```bash
npm install
npm run dev
```

*Generate the package*

```bash
npm run build
npm run pack
```
